Tech Stack

running list

Everything in use across the full coin-poc setup. Useful for explaining the project or onboarding someone.

Smart Contract
Solidity 0.8.20
Contract language — statically typed, compiles to EVM bytecode
OpenZeppelin Contracts
ERC-20, Burnable, Pausable, Ownable — audited base contracts
ERC-20 Standard
Token interface: transfer, approve, allowance, totalSupply, balanceOf
Toolchain
Hardhat
Local EVM node, compile, test, and deploy scripts — runs in Node.js
ethers.js v6
JS library for talking to Ethereum nodes — used in scripts and UI
Mocha + Chai
Test runner and assertion library used by Hardhat's test suite
dotenv
Loads .env vars (private key, RPC URL) into Hardhat config
Blockchain — Linea Mainnet ✓
Linea chosen
ConsenSys / MetaMask zkEVM L2. Chain ID 59144. Very low gas (~$0.10–0.50 to deploy). Strong dev credibility
Lineascan
Block explorer for Linea — lineascan.build. Verify source code after deploy
Linea Bridge
bridge.linea.build — move ETH from Ethereum mainnet to Linea to fund deploy wallet
Dashboard UI
Vanilla HTML/CSS/JS
No build step — open index.html directly or serve statically
ethers.js (CDN)
Same lib as scripts — powers MetaMask connect and contract calls
MetaMask
Browser wallet — injects window.ethereum for signing transactions
AI / Agent Layer
Anthropic SDK @anthropic-ai/sdk
Official Node.js SDK for Claude — used by the PM agent for tool use and prompt caching
Prompt Caching
